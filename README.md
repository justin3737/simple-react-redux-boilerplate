# simple-react-redux-boilerplate
Fast to build a React and Redux template
and deploy to github page.

##Stack

- React.js
- Webpack
- Redux
- Babel
- HMR
- compass 
- scss
- gh-page

##Install npm
``` text
$ npm install
```

##Developement

``` text
$ npm run start
# open this url in your browser
# http://localhost:3000/
```

##Build static files
``` text
$ npm run build
```

##Production

``` text
$ npm run prod
# open this url in your browser
https://<your-github-id>.github.io/<your-repository>
```

##Clean

``` text
$ npm run claen
```
